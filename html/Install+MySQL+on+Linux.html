Linux系統上安裝MySQL

依下達的指令不同，各作業系統的安裝說明如下:

CentOS、Fedora 安裝MySQL

1.請使用yum安裝

#yum install mysql-server

2.啟動MySQL服務

#service mysqld start

3.設置MySQL登入密碼，密碼可自訂，這裡舉例為1qaz

#mysqladmin -u root password 1qaz

4.連線至MySQL

#mysql -uroot -p

Ubuntu、Debian 安裝MySQL

1.請使用apt-get安裝MySQL

#apgt-get install mysql-server

2.啟動MySQL服務

#service mysql start

3.設置MySQL登入密碼，密碼可自訂，這裡舉例為1qaz

#mysqladmin -u root password 1qaz

4.連線至MySQL

#mysql -uroot -p

重設root密碼

若您忘記MySQL的root密碼，可以用以下的方式重設密碼:
1.請先將MySQL服務停止

#service mysqld stop

2.以不檢查密碼的方式啟動MySQL

#mysqld_safe --skip-grant-tables&

3.登入MySQL

#mysql

4.將root密碼重新設定為1234

mysql>update mysql.user set password=PASSWORD('1234') where user='root';

5.執行上一個步驟所下達的密碼設定指令

mysql> flush privileges;

6.退出MySQL

mysql> quit

7.重新啟動MySQL

#service mysqld restart

8.驗證密碼是否已更改

#mysql -uroot -p
