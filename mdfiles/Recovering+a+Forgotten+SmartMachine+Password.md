使用Webmin回復SmartMachine密碼
===
如您忘記SmartMachine只密碼，您可以使用下面方式，透過Webmin來回復您的密碼:


```
https://<PUBLIC_IP>:10000/password-recovery
```

or

```
https://<HOSTNAME>:10000/password-recovery
```
